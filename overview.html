Overview

This is an overview over all my code, published on GitHub and elsewhere.

Apps for Firefox OS

Most of my apps for Firefox are listed on the main page, except for stundenbuch, which is omitted from that list, because it requires a password for access. Most of these apps should also work in any browser.

There are some additional repositories:

  • schnark.github.io contains the code for installation.
  • partial-firefox-marketplace-backup contains a few apps from the old Firefox OS Marketplace.
  • ffos-simulator contains some experimental code to run Firefox OS apps directly in any modern browser.
